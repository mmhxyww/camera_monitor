# Camera Monitor

ブラウザ上で複数のカメラ映像を並べて監視する、シンプルな静的 Web アプリです。
カメラの追加、ラベル編集、並び替え、設定のインポート/エクスポートに対応しています。

## できること

- カメラ URL を追加して一覧表示
- ラベルの編集と削除
- レイアウトを 1 列〜6 列、または自動表示に切り替え
- ドラッグ & ドロップで並び替え
- JSON 形式で設定をインポート/エクスポート
- 接続失敗時の再試行と再読み込み

## 使い方

1. `index.html` をブラウザで開きます。
2. 「ラベル」と「URL」を入力して「追加」を押します。
3. 表示されたタイル上で必要に応じて次の操作を行います。
	- ラベルを直接編集する
	- URL を修正する
	- 右上のボタンで削除する
	- 左上付近のハンドルをドラッグして並び替える
4. 画面上部の `Import` / `Export` で設定を読み込み・書き出しできます。

### インポート時の動作

- JSON ファイル内の有効なカメラ設定を読み込みます。
- 既存の設定がある場合は、置き換えか追加かを確認します。
- 形式が不正な JSON は読み込めません。

### 保存について

- 追加した設定はブラウザの `sessionStorage` に保存されます。
- そのため、同じタブの利用中は復元されますが、ブラウザやタブを閉じると消える場合があります。
- 長期保存したい場合は、必ず `Export` で JSON を保管してください。

## 想定している URL

以下のようなカメラ配信 URL を想定しています。

- MJPEG / MJPG / stream / video / feed 系の配信
- 静止画 URL

静止画 URL は定期的に再取得されます。ストリーム系 URL は、接続維持のため自動再読込が行われます。

## セキュリティ上の注意

- カメラ URL には認証情報や内部ネットワークの情報が含まれることがあります。設定ファイルの取り扱いには注意してください。
- `Export` した JSON は平文です。第三者に渡したり、公開リポジトリへそのまま置いたりしないでください。
- 外部のカメラや不明な配信元を登録すると、映像や URL 情報が意図せず露出する可能性があります。信頼できる環境で利用してください。
- 公開インターネット経由のカメラを扱う場合は、接続先の権限、通信方式、ネットワーク分離を事前に確認してください。

## 補足

- 設定の更新は自動保存されます。
- エクスポートした JSON は、別環境への移行やバックアップに使えます。
- カメラが映らない場合は、URL の形式、ネットワーク到達性、カメラ側の認証設定を確認してください。
